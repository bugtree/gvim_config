1. 用Vim目录中的_vimrc替换安装目录下的_vimrc

2. 将Vim\vimfiles目录中的文件拷贝到安装目录的vimfiles中；
   
   使用pathogen.vim进行插件管理，以后增加插件时，只需要将下载好的插件解压到vimfiles\bundle中即可

3. 若使用ctags与cscope插件，需将cscope目录中的三个文件与ctags目录中的三个文件拷贝到c:\windows\system32\中

4. 若需修改配色，请修改_vimrc与vimfiles\syntax\c.vim
